# Task Management App

Task management app helps users to create/complete tasks and filter them.

## Setup

This application is created by using Vite typescript-react boilerplate.

- Install NodeJS 18 or higher
- Provide `VITE_SERVICE_URL` as `.env` file in case it differs fom the default value
- Run `npm install`
- Run `npm run dev` to run

## Tech Stack

- React
- ReactQuery
- Tailwind
- Formik
- Typescript
